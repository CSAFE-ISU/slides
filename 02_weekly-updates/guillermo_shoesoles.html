Shoeprints Matching

There are several problems related to shoeprints in a crime scene. It is important to distinguish keep in mind what problem is being solved.

  1. Given a shoeprint in a crime scene we want to determine whether it was generated with a given shoe.

  2. Determine the model of a shoeprint. This can be done by looking for the appropriate pattern in a database of shoesoles images.

Processing

It is convenient to invert the colors, since we are going to integrate out values and darker pixels are closer to value zero. Moreover, the each image should be transformed to gray scale.

In Ubuntu:

  1. First, we converted all the tiff files to grayscale using the following command line in Ubuntu’s terminal.

for img in $(find . -iname '*.tiff'); do echo -n "Converting $img"; convert -colorspace GRAY $img $img && echo ' [Done]'; done

  1. Invert the colors of the images to replace the white areas with black.

for img in $(find . -iname '*.tiff'); do echo -n "Inverting $img"; convert -negate $img $img && echo ' [Done]'; done

Pilot Database

There are images of 233 shoe-soles. Several subjects tried different models of shoes and obtained (usually) five replicates of every combination of individual-shoe model-foot. This information is coded in the filename, e.g., “spl_02R_04.tiff” corresponds to the 4th replicate of the right foot of the model 02, which was worn by the subject ‘spl’.
